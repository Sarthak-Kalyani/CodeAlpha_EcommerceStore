# 🛒 CodeAlpha E-Commerce Store

A full-stack E-Commerce Store built using React, Node.js, Express.js, and MongoDB Atlas.

## 🚀 Features

* Product Listing
* Product Images
* Add to Cart
* Remove from Cart
* Total Price Calculation
* Responsive User Interface
* MongoDB Database Integration

## 🛠 Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sarthak-Kalyani/CodeAlpha_EcommerceStore.git
```

### Install Dependencies

Frontend:

```bash
npm install
npm run dev
```

Backend:

```bash
npm install
node server.js
```
